# beojemann.github.io

## General Information

This repository contains the source code for my own personal Quarto website, which was made as part of the ENVS 193DS course in Spring, 2026. The website includes multiple academic projects, research experience, and provides context for my studies of hydrology and geography.

## Data and File Information

- `index.qmd`: Landing page
- `about.qmd`: About Me page with background, academic journey, and hobbies
- `project.qmd`: Projects listing page in blog format
- `fun.qmd`: Personal outdoorsy page with memorable trips and experiences
- `resume.qmd`: Resume page with an embedded PDF
- `custom.scss`: Custom SCSS file for non-default theme colors and non-default fonts
- `_quarto.yml`: The file controlling the entire site such as navbar, theme, and the output
- `posts/ncos-waterbirds/`: Contains the blog post on waterbird-invertebrate trophic network research at NCOS
- `posts/176c-drought/`: Contains the blog post on drought, land-use change, and unemployment project in CA Central Valley
- `posts/115c-tahoe/`: Contains the blog post with project on Sierra Nevada snowpack and Lake Tahoe water quality
- `posts/gee-timeseries/`: Contains the blog post with Python code for land cover change detection using Google Earth Engine
- `images/`: All photos and PDF files used
- `docs/`: Contains the HTML outputs
